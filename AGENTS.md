# The Lazy Blog - Agent Context

This project is a blog built with HTML, CSS, JS (Frontend) and PHP (Backend) using Supabase as the database.
It is designed to be deployed on Vercel using `vercel-php`.

## Environment Variables
The following environment variables are required for the PHP backend to function:
- `SUPABASE_URL`: The URL of the Supabase project.
- `SUPABASE_KEY`: The Service Role Key (for backend operations) or Anon Key.

## Database Schema
Refer to `supabase_schema.sql` for the table structure.

## Deployment
Deploy with `vercel`. Ensure env vars are set in Vercel project settings.
